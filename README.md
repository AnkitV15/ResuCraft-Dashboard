# ResuCraft - Resume Builder

ResuCraft is a modern, web-based resume builder that empowers users to create professional, job-winning resumes in minutes. It provides a user-friendly interface, a variety of templates, and smart features to streamline the resume creation process.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js and npm installed on your machine.

### Backend Repository

The backend for this project is available at [https://github.com/AnkitV15/ResuCraft](https://github.com/AnkitV15/ResuCraft). Please follow the instructions in that repository to set up the backend.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/ResumeBuilder-Dashboard.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Running the Application

To run the app in the development mode:

```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the address shown in your console) to view it in the browser.
